# Remember_words
对市面上背单词功能的优化以及完善（Optimization and improvement of the function of memorizing words on the market）
## 📚 项目简介
一个基于微信小程序的单词记忆工具，支持打卡、词书选择、智能复习、记忆曲线、AI 辅助等功能。

## 📦 技术栈
- 微信小程序原生开发
- 云开发 / Node.js 后端（可切换）
- MongoDB / 云数据库
- Git 版本管理

## 🚀 项目启动
1. 克隆本项目
2. 使用微信开发者工具打开 `miniprogram` 目录
3. 配置测试 AppID 并运行
